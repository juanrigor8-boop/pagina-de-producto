# 🚀 Nexus IA: The Tactical Holographic Projector
**Proyecto: El Lanzamiento del Producto Fantástico** *Curso de Desarrollo Web 2026*

## 📝 Concepto
**Nexus IA** es un proyector holográfico de escritorio diseñado para entusiastas de los juegos de estrategia y cartas coleccionables. El problema que resuelve es la saturación de información en pantallas planas; en su lugar, proyecta un entorno 3D minimalista sobre la mesa de juego, permitiendo visualizar jerarquías de poder, recursos y estadísticas en tiempo real sin perder de vista el tablero físico.

## ⚖️ Optimización de Rendimiento
Para asegurar una carga ultra rápida (siguiendo la "Regla de Oro"):
* **Imágenes:** Todas las capturas fueron convertidas de PNG/JPG a **.webp**, reduciendo el peso de ~2.5MB a solo **380KB** sin pérdida perceptible de calidad.
* **Video:** Optimizado a 1080p con un bitrate balanceado para funcionar como `hero-video` fluido.
* **Lazy Loading:** Implementado en la galería inferior para priorizar el renderizado del Hero Section.
